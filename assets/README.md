# Fortezza Coffee — Brand Assets

## Logo files

- `logo.svg` — full circular badge SVG (cavaliere on horseback, FORTEZZA / KE-KI ON-GA wordmarks). Stylized approximation of the official mark for web use.
- `logo-mark.svg` — small circular F monogram for nav use.
- `favicon.svg` — browser tab favicon.

## Replacing with the official logo

To use the actual Fortezza Coffee logo (the painted cavaliere badge):

1. Save the high-resolution PNG as `assets/logo.png` (transparent background recommended).
2. The hero section in `index.html` will use `logo.png` if present — otherwise it falls back to `logo.svg`.
